# FUTURE_ML_03

## Resume Screening & Ranking System

This project is developed as part of the Machine Learning Track (ML) for Future Interns.

### Project Overview
The Resume Screening System automatically ranks resumes based on their similarity to a given job description using Natural Language Processing (NLP).

### Features
- Reads multiple resumes from a folder
- Reads job description
- Converts text data using TF-IDF Vectorization
- Calculates similarity using Cosine Similarity
- Ranks candidates based on relevance

### Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

### Conclusion
This project demonstrates how Machine Learning and NLP can automate resume screening and improve recruitment efficiency.
